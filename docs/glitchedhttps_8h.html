glitchedhttps_response* glitchedhttps_submit ( const glitchedhttps_request *  request)

Submits a given HTTP request and returns the server response.

This allocates memory, so don't forget to glitchedhttps_response_free() the returned glitchedhttps_response instance after usage!!

Parameters
requestThe glitchedhttps_request instance containing the request parameters and data (e.g. url, body, etc...).
Returns
The (freshly allocated) glitchedhttps_response instance containing the response headers, status code, etc... if the request was submitted successfully; NULL if the request couldn't even be submitted (e.g. invalid URL/server not found/no internet/whatever..).
